This project is to implement the features of Cash Register. It has options for purchase and displaying inventory/stock and purchase history.

It comprises 4 files. 1)Store.java 2)Stock.java 3)StoreItem.java 4)PurchaseHistory.java

Store.Java has main() method; so application starts by executing store.java.

StoreItem.java and PurchaseHistory.java both are model class files of our project, Store.java is view and Stock.java is controller of our project in MVC architecture of project.

For Manager option; please use password "mgr123" when prompted. Then only the options under manager to add stock, disply stock, display purchase history are accessed.
